Privacidade e Proteção de Dados

Política de Privacidade

Esta política explica como o Dosiq trata dados pessoais no app web e no app mobile, com foco em transparência, minimização de dados e cuidado com informações de saúde.

Produto: Dosiq Versão do documento: 0.1 Última atualização: 14/04/2026

1. Introdução

O Dosiq é um aplicativo para organização da rotina de medicamentos, acompanhamento de tratamentos, registro de doses e controle de estoque.

Esta Política de Privacidade explica quais dados podem ser tratados pelo aplicativo, para quais finalidades, como esses dados são protegidos e quais escolhas e direitos o usuário possui.

2. Responsável pelo tratamento dos dados

Responsável pelo aplicativo: Dosiq

Contato de suporte e privacidade: contact@coelho.me

3. Quais dados podem ser tratados

3.1. Dados de cadastro e autenticação

  • e-mail
  • identificador interno de usuário
  • informações necessárias para autenticação e manutenção de sessão

3.2. Dados de saúde e rotina informados pelo usuário

  • nomes de medicamentos
  • tratamentos e protocolos cadastrados
  • doses, horários e frequências
  • registros de doses tomadas
  • informações de estoque dos medicamentos

Esses dados podem ser considerados dados pessoais sensíveis, nos termos da LGPD, por estarem relacionados à saúde.

3.3. Dados técnicos mínimos de funcionamento

  • informações básicas de sessão
  • identificadores técnicos necessários para funcionamento da conta
  • dados necessários para segurança, autenticação e integridade da aplicação

3.4. O que o app não faz por padrão

  • não vende dados pessoais
  • não compartilha dados de saúde com anunciantes
  • não usa rastreamento publicitário como finalidade principal do produto

4. Para que usamos esses dados

  • permitir login e acesso seguro à conta
  • mostrar a rotina diária de medicamentos
  • registrar doses e acompanhar adesão ao tratamento
  • exibir tratamentos ativos
  • controlar estoque de medicamentos
  • manter sincronização entre dispositivos, quando aplicável
  • oferecer suporte ao usuário
  • cumprir obrigações legais e de segurança, quando necessário

O app não substitui acompanhamento médico, diagnóstico, prescrição ou atendimento de urgência.

5. Base legal e princípios de privacidade

O tratamento de dados pessoais é realizado com fundamento nas bases legais aplicáveis, incluindo execução dos serviços solicitados pelo usuário, segurança da aplicação, cumprimento de obrigações legais e, quando necessário, consentimento para tratamentos específicos.

O Dosiq busca seguir os princípios da LGPD, especialmente:

  • finalidade
  • necessidade
  • transparência
  • segurança
  • prevenção

6. Armazenamento e infraestrutura

O Dosiq pode utilizar infraestrutura própria e de terceiros para operar a aplicação, incluindo autenticação, banco de dados, hospedagem e serviços necessários ao funcionamento do produto.

No estado atual do projeto, parte central da persistência e autenticação é operada com apoio de infraestrutura Supabase.

7. Compartilhamento de dados

Os dados não são compartilhados com terceiros para venda de informações pessoais.

O compartilhamento pode ocorrer apenas nas seguintes hipóteses:

  • com provedores de infraestrutura necessários para operar o aplicativo
  • quando houver obrigação legal ou ordem de autoridade competente
  • quando o usuário solicitar ou autorizar uma integração específica

8. Segurança das informações

São adotadas medidas razoáveis para proteger os dados pessoais contra acesso não autorizado, uso indevido, alteração, divulgação ou destruição indevida.

  • autenticação de usuário
  • controle de acesso por conta
  • segregação lógica de dados
  • uso de infraestrutura com mecanismos adequados de segurança

Ainda assim, nenhum sistema é totalmente invulnerável. Por isso, também é importante que o usuário proteja suas credenciais de acesso.

9. Direitos do usuário

Nos termos da legislação aplicável, o usuário pode solicitar:

  • confirmação da existência de tratamento
  • acesso aos próprios dados
  • correção de dados incompletos, inexatos ou desatualizados
  • anonimização, bloqueio ou eliminação, quando cabível
  • portabilidade, quando aplicável
  • informações sobre compartilhamento
  • revogação de consentimento, quando essa for a base legal aplicável
  • solicitação de exclusão de conta e dados, conforme limites legais e operacionais

10. Exclusão de conta e dados

No período de MVP e beta, o processo de exclusão de conta pode ocorrer inicialmente por solicitação ao suporte.

Canal de solicitação: contact@coelho.me

11. Retenção de dados

Os dados podem ser mantidos pelo tempo necessário para:

  • executar as funcionalidades do app
  • atender solicitações do usuário
  • cumprir obrigações legais e regulatórias
  • preservar segurança e integridade do serviço

12. Crianças e adolescentes

O Dosiq não é direcionado intencionalmente a crianças sem supervisão de responsável. Se o uso envolver dados de menores de idade, devem ser observados os requisitos legais aplicáveis.

13. Transferência internacional

Dependendo dos provedores de infraestrutura utilizados, alguns dados podem ser processados em servidores localizados fora do Brasil. Quando isso ocorrer, serão buscados fornecedores com padrões adequados de segurança e proteção de dados.

14. Alterações desta política

Esta Política de Privacidade poderá ser atualizada periodicamente para refletir evolução do produto, mudanças legais ou alterações técnicas relevantes. A versão atualizada passará a ser disponibilizada no canal oficial do produto.
