# R-166: UX Language Parity — Mobile Must Mirror Web (P-011)

**Criado:** 2026-04-14 | **Origem:** Maintainer decision após detectar "Registar" (pt-EU) no mobile

---

## O Problema

Componentes mobile usavam português europeu ("Registar", "registados") enquanto a web usa pt-BR coloquial ("Tomar", "tomados"). Isso cria dissonância cognitiva para utilizadores multiplataforma.

## A Regra

Antes de escrever **qualquer texto visível ao utilizador** em componente mobile:

1. Localizar o componente web equivalente (`grep -rn "keyword" src/features/ src/shared/`)
2. Copiar a linguagem exacta
3. Se o componente não existir na web, seguir o glossário P-011

## Glossário P-011

| Contexto | ✅ pt-BR (paridade web) | ❌ Incorrecto |
|----------|------------------------|--------------|
| CTA dose | **Tomar** | Registar, Registrar |
| Status dose | **Tomada / Tomados** | Registada, Registados |
| Modal dose | **Tomar dose** | Registar dose |
| Dose perdida | **Perdida** | Em falta |
| Dose agendada | **Agendada** | Programada |
| Completo | **✓ Feito** | ✓ Completo |
| Ação stock | **Adicionar** | Registar entrada |
| Erro interno | **Erro ao registrar** | Erro ao registar |
