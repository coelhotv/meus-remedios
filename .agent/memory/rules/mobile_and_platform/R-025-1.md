# R-025-1 — Reuse Shared Patient Identity Helpers [MEDIUM]

## Rule
When any feature needs patient display fallback or email-handle extraction, reuse `src/shared/utils/patientUtils.js` instead of re-implementing local helpers.

## Why
PR #422 review follow-up (2026-03-25). Local re-implementations of the same logic caused inconsistent labels across consultation and report flows.

`src/shared/utils/patientUtils.js`
