# R-155: PriorityDoseCard registra TODOS os doses da faixa

**Summary:** PriorityDoseCard recebe doses={urgentDoses} sem slice. Display limita a 3 internamente (DISPLAY_LIMIT). CTA 'Confirmar Agora' registra TODOS, não apenas os visíveis.

**Applies to:** react

**Tags:** react, safety

## Rationale

Display limits must not become action limits. The user-facing CTA has to honor the full intended batch.

## Implementation Notes

Pass the full dose collection into `PriorityDoseCard` and let the component handle visual truncation only.
