# R-148: Domain-Aware Case Conversion for Autocomplete Data

**Summary:** Apply case conversion based on semantic meaning, not one-size-fits-all

**Applies to:** ts, react

**Tags:** design, react

## Rationale

Different fields serve different purposes. Names need Title Case; metadata needs Sentence Case. Domain-aware conversion improves UX.
