# [R-183] Threshold de Complexidade Adaptativa (Mobile)

## Descrição
Para garantir a melhor experiência de usuário baseada na carga medicamentosa, o aplicativo mobile deve alternar entre dois modos de visualização na aba Hoje.

## Gatilho
Utilizado ao renderizar a agenda diária no Dashboard.

## Regra
- **Threshold**: Se o usuário possui `<= 3` medicamentos ativos, o modo é `Simple` (Dona Maria).
- **Threshold**: Se o usuário possui `4` ou mais medicamentos ativos, o modo é `Complex` (Carlos).
- No modo `Simple`, as doses são exibidas em uma lista única cronológica, sem agrupamentos por turno ou accordions.
- No modo `Complex`, as doses devem ser agrupadas por turnos com suporte a colapso/expansão.

## Exemplo
```javascript
const medicinesCount = Object.keys(data.medicines).length
const isComplex = medicinesCount > 3
```
