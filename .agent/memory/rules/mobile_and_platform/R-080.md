# R-080 — Cache Architecture Optimizations [REFERENCE]

## Status
All OOM issues FIXED as of 2026-02-23.

## Fixes applied to queryCache.js
1. **Generation Counter** — prevents background revalidation from writing after `clearCache()`
2. **TTL Eviction** — GC purges entries older than 2× STALE_TIME (60s)
3. **Debounced Persistence** — `localStorage` writes debounced to 500ms (was synchronous)
4. **Access Counter Reset** — `accessCounter` reset in `clearCache()` for correct LRU
5. **Render Loop Fix** — `useCachedQueries` uses stable `queriesKey` (not `[queries]` array)
6. **Code Deduplication** — extracted `executeParallelQueries()` to eliminate `fetchAll`/`refetchAll` duplication

All 26 test files pass on 8GB machines with `npm run test:lowram`.
