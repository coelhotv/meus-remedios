# R-113: Filter Logs by protocol_id Only, Never by medicine_id

**Summary:** Filter logs using ONLY log.protocol_id === protocolId. Never add OR with medicine_id — 2+ protocols may share the same medicine.

**Applies to:** all

**Tags:** data, supabase, safety

## Rationale

The same medicine can appear in multiple protocols. Filtering by medicine ID causes cross-protocol contamination.

## Implementation Notes

Treat `protocol_id` as the canonical identifier for protocol-scoped adherence and logging calculations.
