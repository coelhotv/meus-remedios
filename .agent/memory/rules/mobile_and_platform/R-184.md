# [R-184] Protocolo de Auto-Refresh de Meia-Noite

## Descrição
Garante que a agenda de medicamentos seja atualizada automaticamente quando o dia vira no fuso horário local do dispositivo, sem exigir que o usuário feche/abra o app ou faça pull-to-refresh.

## Gatilho
Hooks de dados do Dashboard (`useTodayData`).

## Regra
1. **Midnight Timer**: Calcular os milissegundos restantes até a próxima meia-noite local e agendar um `setTimeout` para disparar o `load()`.
2. **AppState Listener**: Adicionar um listener para o estado `active`. Ao retornar do segundo plano (background -> active), verificar se a data local mudou (`data.localDay !== getTodayLocal()`).
3. Se o dia mudou, disparar `load()` imediatamente.

## Exemplo
```javascript
useEffect(() => {
  const scheduleRefresh = () => {
    const msUntilMidnight = calculateMilliseconds();
    setTimeout(() => { load(); scheduleRefresh(); }, msUntilMidnight);
  }
  scheduleRefresh();
  
  const sub = AppState.addEventListener('change', (state) => {
    if (state === 'active' && dayChanged) load();
  });
  return () => sub.remove();
}, [load]);
```
