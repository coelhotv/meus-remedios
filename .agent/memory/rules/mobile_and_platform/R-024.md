# R-024 — Daily PDF Adherence Must Count Dose Events [HIGH]

## Rule
In the clinical PDF daily table:
- Count **completed vs expected dose events** for each day
- Exclude future `scheduledDoses`
- Do NOT reuse monthly aggregate totals per row
- Do NOT switch to pill-quantity math unless UI and labels are redesigned

## Why
PR #422 review follow-up (2026-03-25). Using monthly aggregates per row produced mathematically wrong daily counts in the PDF.
