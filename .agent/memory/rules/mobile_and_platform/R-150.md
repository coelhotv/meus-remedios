# R-150: Compute State-Specific Aggregations, Not Just Primary State

**Summary:** Compute groups/filtered lists for ALL states, not just the primary

**Applies to:** ts, react

**Tags:** react, performance

## Rationale

Lazy computation by state causes missing data bugs. Compute for all states to prevent tab-switching data loss.
