# [R-182] Cross-Platform Build Version Synchronization

## Context
Google Play uses `versionCode` (integer) and App Store uses `buildNumber` (string) to identify binary versions. Manual management leads to drift and confusion.

## Rule
Maintain absolute version consistency by deriving both platform-specific identifiers from the semantic `APP_VERSION` in `app.config.js`.
- Use the formula: `major * 10000 + minor * 100 + patch`.
- Apply to `android.versionCode` as a number.
- Apply to `ios.buildNumber` as a string.

## Rationale
Ensures that a single source of truth (`APP_VERSION`) updates all platforms simultaneously, simplifying support and release tracking.
