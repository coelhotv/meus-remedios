# R-149: Full Object Fetch for Edit Workflows

**Summary:** Fetch complete entity before opening edit forms

**Applies to:** ts, supabase

**Tags:** supabase, react

## Rationale

Aggregated data is incomplete. Full entity fetch ensures all required fields are present and schema is correct.
