# R-115: React Virtuoso for Long Lists on Mobile

**Summary:** Lists with 30+ items on mobile should use react-virtuoso instead of .map(). Configure: useWindowScroll, overscan=300, endReached. Wrap items with React.memo.

**Applies to:** react

**Tags:** react, performance

## Rationale

Long mapped lists degrade mobile performance quickly. Virtualization keeps rendering cost proportional to what is visible.

## Implementation Notes

Use `react-virtuoso` for long mobile lists and keep item rendering memoized where appropriate.
