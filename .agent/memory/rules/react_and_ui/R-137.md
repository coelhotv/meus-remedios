# R-137: Font Weight Accessibility

**Summary:** NUNCA use font-weight < 400 (thin/light). Pacientes idosos não conseguem ler fontes finas. Mapear light=400, thin=400.

**Applies to:** all

**Tags:** design

## Rationale

Very thin fonts reduce readability for the actual user profile of the product.

## Implementation Notes

Treat 400 as the minimum accessible weight for production UI copy.
