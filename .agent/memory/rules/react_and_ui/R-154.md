# R-154: Reutilizar StockPill para qualquer status de estoque

**Summary:** Use shared StockPill component for all stock status badges

**Applies to:** react

**Tags:** design, react

## Rationale

Consistency between Treatments and Stock screens is critical. Two badge systems confuse users. Always import shared StockPill.
