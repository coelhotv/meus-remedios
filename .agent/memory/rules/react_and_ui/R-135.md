# R-135: Prefer Direct Actions Over Modal Flows for Common Operations

**Summary:** Use 1-click direct actions for simple operations; reserve modals for complex workflows

**Applies to:** react

**Tags:** design, react

## Rationale

User feedback: TOMAR AGORA button should register dose immediately, not open modal. Modal overhead hurts UX for frequent operations.
