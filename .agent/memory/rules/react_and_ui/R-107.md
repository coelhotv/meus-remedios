# R-107: useState não Reage a Mudanças de Prop Derivada — Usar useEffect

**Summary:** useState(() => derivedValue) usa o initializer APENAS na montagem. Se derivedValue pode mudar, adicionar useEffect para sincronizar.

**Applies to:** react

**Tags:** react

## Rationale

State initializers are not reactive. Assuming they follow future prop changes creates stale UI and silent logic mismatches.

## Implementation Notes

Use `useEffect` or remount strategy when derived state must stay aligned with changing props.
