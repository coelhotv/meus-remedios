# R-109: key={controllingProp} para Reinicializar Estado Interno

**Summary:** Quando estado interno é inicializado com base em prop e deve reinicializar quando a prop muda, usar key={controllingProp} no JSX para forçar remount.

**Applies to:** react

**Tags:** react

## Rationale

Some internal state should reset when a controlling prop changes. Using `key` is the explicit remount mechanism when sync logic would be more brittle.

## Implementation Notes

Apply `key={controllingProp}` to the component boundary when full internal reset is the intended behavior.
