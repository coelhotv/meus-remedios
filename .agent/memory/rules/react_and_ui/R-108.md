# R-108: Page Visibility API para Pausar Intervals em Abas Inativas

**Summary:** setInterval continua em abas não visíveis. Usar visibilitychange para pausar/retomar e atualizar estado ao voltar à aba.

**Applies to:** react

**Tags:** performance, react

## Rationale

Background intervals waste work and can cause stale state when the user returns to the tab.

## Implementation Notes

Use the Page Visibility API to pause unnecessary polling or interval-based updates while the page is hidden.
