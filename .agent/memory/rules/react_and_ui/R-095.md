# R-095: Wave 1 Component Purity — No Context Imports

**Summary:** All Wave 1 (Onda 1) components must receive data ONLY via props. Never import useDashboardContext or any feature context in Onda 1.

**Applies to:** react

**Tags:** react, design

## Rationale

Wave 1 components were designed to stay presentational and reusable. Pulling feature context into them breaks that boundary and makes later extraction harder.

## Implementation Notes

Pass all needed data and handlers through props. Keep feature context usage in higher orchestration layers.
