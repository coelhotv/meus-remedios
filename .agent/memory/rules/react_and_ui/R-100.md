# R-100: Multiple useDashboard() Calls are Safe

**Summary:** Multiple components calling useDashboard() is safe. React Context consumers always receive the same object reference — no performance penalty.

**Applies to:** react

**Tags:** react, performance

## Rationale

This rule prevents unnecessary refactors motivated by a false optimization concern. Context consumption here is not itself the problem.

## Implementation Notes

Do not collapse or memoize component structure solely to avoid multiple `useDashboard()` calls.
