# R-102: SwipeRegisterItem onRegister vs onRegisterDose Mismatch

**Summary:** SwipeRegisterItem calls onRegister(medicineId, dosage). DoseZoneList needs onRegisterDose(protocolId, dosage). Use closure wrapper.

**Applies to:** react

**Tags:** react

## Rationale

The callback signatures are not interchangeable. A wrapper preserves both component contracts and keeps the mismatch localized.

## Implementation Notes

Adapt the required arguments with a closure at the integration point.
