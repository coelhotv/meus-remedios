# R-118: Redesign CSS Color System

**Summary:** All color values in components.redesign.css MUST come from CSS variables (--color-*). Never hardcode hex or rgba. Use color-mix() for derived variations.

**Applies to:** all

**Tags:** design

## Rationale

Hardcoded colors fragment the design system and make redesign changes expensive. CSS variables keep visual decisions centralized.

## Implementation Notes

Use only canonical color tokens in redesign styles. Derived variants should still be built from tokens.
