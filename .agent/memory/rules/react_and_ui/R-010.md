# R-010 — Hook Declaration Order [CRITICAL]

## Rule
Always declare React hooks in this order:
1. States (`useState`)
2. Memos (`useMemo`)
3. Effects (`useEffect`)
4. Handlers (functions)

Declaring state AFTER `useMemo` causes TDZ (Temporal Dead Zone) ReferenceError.

## Why
Dashboard crash from `snoozedAlertIds` TDZ — `useMemo` referenced state before it was declared.

## Examples
```jsx
// Correct
const [data, setData] = useState()
const processed = useMemo(() => transform(data), [data])
useEffect(() => { fetch() }, [processed])
const handleClick = () => { ... }

// Wrong — ReferenceError
const processed = useMemo(() => data + 1, [data]) // data undefined!
const [data, setData] = useState(0)
```
