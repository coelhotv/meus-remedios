# R-153: Dona Maria quer hierarquia, não menos dados

**Summary:** Remove data only if it doesn't inform action, never remove by reflex

**Applies to:** react

**Tags:** design

## Rationale

Simple mode still needs context. Translate data to human language instead of hiding. Verify: CTA dominant, numeric data has human equivalent, no forced comparison, no redundancy.
