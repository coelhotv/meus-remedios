# R-156: AdherenceLabel usa linguagem humana (não %) para modo simple

**Summary:** Em isComplex=false, usar AdherenceLabel (não AdherenceBar7d). Thresholds: >90% 'Tratamento em dia', 70-90% 'Algumas doses perdidas', 50-70% 'Tratamento em risco', <50% 'Muitas doses perdidas'.

**Applies to:** react

**Tags:** react, design

## Rationale

Simple-mode users respond better to human-language interpretation than to raw percentages.

## Implementation Notes

Translate adherence thresholds into human-readable labels when rendering the simple mode.
