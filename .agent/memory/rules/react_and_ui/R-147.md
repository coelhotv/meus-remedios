# R-147: Framer Motion Cascade Re-Triggering via Key Remounting

**Summary:** Use key prop on containers to force animation re-trigger on data changes

**Applies to:** react

**Tags:** design, react

## Rationale

Cascade animations don't re-trigger when data changes unless component remounts. Key prop forces remount and re-animation.
