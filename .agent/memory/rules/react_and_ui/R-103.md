# R-103: Props Passadas mas não Declaradas na Assinatura São Silenciosas

**Summary:** Uma prop passada no JSX mas ausente no destructuring da função é silenciosamente ignorada — sem erro em runtime. Listar TODAS as props interativas no destructuring.

**Applies to:** react

**Tags:** react, safety

## Rationale

React does not warn when a passed prop is simply not destructured. This creates silent feature failures that look like logic bugs elsewhere.

## Implementation Notes

Whenever a component gains a new interactive prop, update the function signature explicitly.
