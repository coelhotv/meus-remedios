# R-105: Remover Dead Code Imediatamente ao Substituir Seções

**Summary:** Ao substituir uma seção JSX por novo componente, remover simultaneamente: props antigas, useMemos/states relacionados, imports não usados.

**Applies to:** react

**Tags:** react, process

## Rationale

Leaving old support code behind creates ambiguity about what is still active and makes future edits riskier.

## Implementation Notes

Treat component replacement as a full cleanup step, not just a JSX swap.
