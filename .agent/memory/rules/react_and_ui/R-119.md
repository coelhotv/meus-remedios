# R-119: Button Size Class Preservation

**Summary:** min-height and padding MUST be ONLY in size classes (.btn-sm, .btn-md, .btn-lg). Never put in variant classes (.btn-primary, .btn-danger).

**Applies to:** all

**Tags:** design

## Rationale

Mixing size and visual variant responsibilities makes buttons harder to reason about and easier to break during overrides.

## Implementation Notes

Keep sizing in size classes and visual treatment in variant classes.
