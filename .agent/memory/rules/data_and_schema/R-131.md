# R-131: parseLocalDate em TODAS as Queries de Data ao Supabase

**Summary:** Queries Supabase com filtros de data DEVEM converter datas locais para UTC via parseLocalDate(). NUNCA usar template T00:00:00.000Z — ignora fuso horário.

**Applies to:** supabase

**Tags:** supabase, timezone, safety

## Rationale

UTC templates built from local dates skip timezone conversion and cause off-by-one-day bugs.

## Implementation Notes

Use `parseLocalDate()` as the canonical path for local-date conversion before Supabase filtering.
