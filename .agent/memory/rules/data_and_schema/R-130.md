# R-130: Zod Validation em TODOS os Métodos de Leitura de Services

**Summary:** Não validar apenas métodos de escrita. Métodos de leitura (get/list/getByRange) também DEVEM ter schemas Zod validando parâmetros.

**Applies to:** zod

**Tags:** zod, data

## Rationale

Read paths also accept inputs and can fail silently if parameters are malformed. Validation coverage must include both read and write methods.

## Implementation Notes

Apply Zod validation at read-service boundaries just as strictly as in mutation methods.
