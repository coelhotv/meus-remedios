# R-122: Extract Functions at 30-Line Threshold

**Summary:** If a function exceeds 30 lines, extract pure helpers. Functions >30 lines are harder to test, understand, and reuse.

**Applies to:** all

**Tags:** process

## Rationale

Long functions accumulate branching and hidden responsibilities. Extraction improves readability and testability.

## Implementation Notes

Prefer pure helper extraction when a function starts growing past the threshold.
