# R-075 — Low-RAM Test Compatibility [CRITICAL for MacBook Air 2013]

## Rule
All tests must pass on 8GB RAM machines using `npm run test:lowram`.

Split large test files (>300 lines) into smaller files by test scope. Each file = fresh worker = no memory accumulation.

## Why
2026-02-23 low-RAM optimization sprint. 8GB machines (MacBook Air 2013) were failing test runs due to OOM. Sequential execution in separate workers prevents memory accumulation across tests.

## Command
```bash
npm run test:lowram  # Sequential, excludes memory-intensive tests
```
