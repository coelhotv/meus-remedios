# R-061 — Mandatory Review Pause [HIGH]

## Rule
Always have a pause for review between coding tasks. Never create sequential tasks without validation gates. Quality checkpoints must exist between implementation phases.

## Why
2026-W09 coordination failure. Sequential tasks without review pauses created a cascade of unreviewed changes that compounded into a production incident.
