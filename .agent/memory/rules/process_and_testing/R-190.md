# R-190 — Mandatory Devflow Bootstrap (Hard Stop)

## Regra
É terminantemente PROIBIDO para qualquer agente de IA iniciar a escrita de código ou modificação de arquivos sem antes executar o protocolo de **Bootstrap (Fase 0)** do `/devflow`.

**Protocolo de Sessão:**
1. Nova Sessão detectada → **Hard Stop**.
2. Executar `/devflow` para carregar `state.json` e índices de memória.
3. Verificar o `goal` da sprint atual.
4. Somente após a confirmação do contexto, proceder com a tarefa.

## Por que?
Sem o bootstrap, o agente opera em "modo cego" em relação ao estado atual do projeto (Wave, PRs pendentes, Sprint Goal), o que leva a alucinações sobre versões de iOS (ex: ignorar iOS 26) ou regressões de lógica que já foram discutidas em sessões anteriores.

## Sanção
O descumprimento desta regra invalida a sessão de desenvolvimento e requer um reset imediato do agente para o ponto de sincronização estável.
