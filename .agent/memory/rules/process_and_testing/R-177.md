# R-177: Always use ~/git-icloud/meus-remedios/ as working path

**Daqui para frente, usar o alias `~/git-icloud/meus-remedios/` para o projeto.**

Os agentes e sessões DEVFLOW DEVEM usar primariamente o caminho `~/git-icloud/meus-remedios/` para interagir com este projeto. 
Evite usar o caminho original extenso de iCloud (`~/Library/Mobile Documents/com~apple~CloudDocs/...`) para prevenir problemas sistêmicos, de Watchman, ou ferramentas que quebram com espaços nos nomes de diretórios.
