# R-093: ESLint Unused Vars — Remove, Don't Alias

**Summary:** When a destructured prop/variable is unused, remove it entirely. Do NOT use _prefix aliasing — varsIgnorePattern doesn't reliably cover aliased destructuring.

**Applies to:** all

**Tags:** process

## Rationale

Aliasing unused values hides dead code instead of removing it. The safest outcome is to delete the unused binding and keep the signature honest.

## Implementation Notes

If a destructured prop or variable is unused, remove it from the declaration rather than masking it.
