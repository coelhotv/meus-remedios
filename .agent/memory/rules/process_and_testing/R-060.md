# R-060 — No Self-Merge for Code Agents [CRITICAL]

## Rule
Never allow a code agent to merge its own PR.

**Required flow:** Code agent creates PR → another agent or human reviews → DevOps agent merges ONLY after explicit approval.

## Why
journal/2026-W09 coordination failure. A code agent merged unreviewed code, introducing bugs that took 2 days to diagnose. Self-merge removes the only quality gate.
