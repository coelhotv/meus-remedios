# R-076 — Disable localStorage in Tests [HIGH]

## Rule
Tests must disable localStorage persistence. The queryCache must check `process.env.NODE_ENV === 'test'` and skip `localStorage.setItem()` / `localStorage.getItem()`.

## Why
2026-02-23 queryCache optimization. localStorage writes in tests accumulated ~200MB per test run on 8GB machines, causing OOM failures.

## Pattern
```javascript
// CORRECT — skip in test env
function persistCache() {
  if (process.env.NODE_ENV === 'test') return
  localStorage.setItem(CACHE_CONFIG.PERSIST_KEY, JSON.stringify(entries))
}

// WRONG — accumulates memory
function persistCache() {
  localStorage.setItem(CACHE_CONFIG.PERSIST_KEY, JSON.stringify(entries))
}
```

- `AP-T03` — store data in localStorage during tests
