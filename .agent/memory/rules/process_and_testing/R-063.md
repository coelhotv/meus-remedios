# R-063 — Process as Protection [MEDIUM]

## Rule
The rigorous workflow exists to prevent errors. Never skip steps "just this once." Process violations compound into production failures.

## Why
2026-W09. Each "just this once" exception created a precedent. By the time the incident occurred, the team had normalized skipping 3 separate safety steps.
