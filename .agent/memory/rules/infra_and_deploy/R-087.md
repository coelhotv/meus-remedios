# R-087 — Structured Logging from Day One [HIGH]

## Rule
Implement structured logging (timestamp, level, context, correlationId) from the first commit of any API endpoint.

## Pattern
```javascript
export function log(endpoint, level, message, data = {}) {
  const logEntry = {
    timestamp: new Date().toISOString(),
    endpoint,
    level,
    message,
    correlationId: getCorrelationId(),
    ...data
  }
  console.log(JSON.stringify(logEntry))
}
```

## Source
Sprint 7 (PR #148) — hours wasted debugging without logs

## Consolidated anti-patterns
- `AP-S06` — ship API code without structured logging

Use this rule as the canonical guidance for logging discipline in endpoints and handlers. The AP remains as historical shorthand.
