# R-091 — Underscore Prefix for API Utilities [HIGH]

## Rule
Helper files, shared utilities, and extracted handler functions inside `api/` MUST be in directories prefixed with `_` (underscore). Without prefix, Vercel counts them as serverless functions.

## Correct Patterns
- `api/gemini-reviews/_shared/logger.js` — NOT counted
- `api/gemini-reviews/_handlers/persist.js` — NOT counted
- `api/dlq/_handlers/retry.js` — NOT counted

## Wrong Patterns
- `api/gemini-reviews/shared/logger.js` — COUNTED
- `api/utils.js` — COUNTED

## Source
api/gemini-reviews/shared/logger.js and security.js were counted as functions, wasting 2 of 12 budget

## Consolidated anti-patterns
- `AP-S11` — place helper files in non-underscored API paths

Use this rule as the canonical guidance for Vercel function-budget hygiene. The AP remains as a negative lookup.
