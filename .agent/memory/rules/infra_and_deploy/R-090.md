# R-090 — Vercel Hobby 12-Function Limit [CRITICAL]

## Rule
Vercel Hobby plan allows max 12 serverless functions. Every `.js` file in `api/` counts (including subdirectories), UNLESS in a `_`-prefixed or `.`-prefixed directory.

**Before creating ANY new `.js` in `api/`, verify budget:**

```bash
find api -name "*.js" -not -path "*/_*" -not -path "*/.*" -not -name "CLAUDE.md" | wc -l
```

If >= 10: consolidate into existing router instead of creating new file.

6/12 functions used (post-consolidation), 6 slots free.
