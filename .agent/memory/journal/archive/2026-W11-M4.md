## Learnings

- `useSyncExternalStore` é o hook correto para subscrever em eventos externos (online/offline, localStorage, etc.) — evita `useState + useEffect` com lógica de cleanup manual
- Sempre verificar tokens CSS existentes antes de criar variáveis novas (`--color-text-on-warning` não existia)
- O projeto tem `src/shared/styles/tokens/z-index.css` com `--z-toast: 800` — usar sempre ao invés de hardcode
