# ADR-018: Stock Levels System: Four-Tier Threshold Model

**Status:** Accepted | **Date:** 2026-03-27 | **Source:** PR #433
**Tags:** stock, inventory, schema

## Context
Users need clear visual indicators for medication stock levels to know when to order refills.

## Decision
Implement four-tier stock level system with thresholds and visual colors:

| Level | Threshold | Color | Icon | Label |
|-------|-----------|-------|------|-------|
| CRITICAL | < 7 days | #ef4444 (red) | AlertCircle | Crítico |
| LOW | < 14 days | #f59e0b (orange) | AlertTriangle | Baixo |
| NORMAL | < 30 days | #22c55e (green) | CheckCircle | Normal |
| HIGH | ≥ 30 days | #3b82f6 (blue) | TrendingUp | Bom |

## Implementation
- Calculate days_remaining = quantity / daily_dosage
- Use thresholds to determine level
- Display in UI with color and icon
- Alert on transitions to CRITICAL/LOW

## Consequences
✅ Clear visual feedback
✅ Data-driven refill decisions
⚠️ Accurate daily dosage calculation required
⚠️ Must handle edge cases (variable dosage)
