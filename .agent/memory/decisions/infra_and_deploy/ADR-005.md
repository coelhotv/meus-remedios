# ADR-005: Add Therapeutic Class Field to Medicines

**Status:** Accepted | **Date:** 2026-03-21 | **Source:** PR #415
**Tags:** medicines, schema, feature

## Context
Clinical users requested ability to categorize medicines by therapeutic class (e.g., "Antihypertensive", "Antibiotic") for better organization and clinical decision support.

## Decision
Add `therapeutic_class` field to medicine schema with predefined categories matching Anvisa standards.

## Implementation
- Add to Zod schema: `therapeutic_class: z.enum(['Antihypertensive', 'Antibiotic', ...])`
- Add to Supabase medicines table
- Update form with therapeutic class selector
- Implement search/filter by therapeutic class

## Consequences
✅ Better medicine organization
✅ Clinical decision support
⚠️ Schema migration required
⚠️ May require data backfill for existing medicines
