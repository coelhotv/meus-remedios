# ADR-011: Typography Stack: Public Sans (Display) + Lexend (Body)

**Status:** Accepted | **Date:** 2026-03-24 | **Source:** PR #418
**Tags:** typography, design, fonts

## Context
Need modern, readable typography suitable for medical information display with good accessibility.

## Decision
Use **Public Sans** for displays/headlines (geometric, modern) and **Lexend** for body/labels (dyslexic-friendly, readable).

## Implementation
- Load via `@import url(...)` in CSS (no HTML link tags)
- Public Sans: display, headlines, navigation
- Lexend: body text, labels, small text

## Consequences
✅ Excellent readability
✅ Dyslexic-friendly option (Lexend)
✅ Modern geometric aesthetic
⚠️ Two font families (slightly larger bundle)
⚠️ Font loading may affect FCP
