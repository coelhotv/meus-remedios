# ADR-010: Gradual Color Palette Transition (Neon → Health-Focused)

**Status:** Accepted | **Date:** 2026-03-24 | **Source:** PR #417
**Tags:** design, colors, branding

## Context
Original cyberpunk/neon aesthetic doesn't align with healthcare mission. Need warm, trustworthy color scheme.

## Decision
Transition primary colors:
- Primary: Neon #00ff00 → Health green #006a5e
- Secondary: Neon #ff00ff → Clinical blue #005db6
- Implement Material Design 3 surface hierarchy

## Implementation
```css
--color-primary: #006a5e;      /* Health/wellness */
--color-secondary: #005db6;    /* Trust/clinical */
--color-error: #ef4444;        /* Alert red */
--color-surface: #f8fafb;      /* Clean background */
```

## Consequences
✅ Better brand alignment
✅ Improved accessibility (neon colors are hard to read)
⚠️ Complete design overhaul
⚠️ User may need time to adjust
