# [AP-055] App Store Connect Manual Binary Upload Myth

## Context
Developers coming from the Google Play Console ecosystem often expect to find a manual "Upload" button for the `.ipa` (binary) file in the App Store Connect web interface.

## Anti-Pattern
Searching for a manual upload button in the App Store Connect browser dashboard to upload a production binary.

## Why this is a problem
- **Time Waste:** The button does not exist. Apple only accepts binaries via authorized transport tools.
- **Confusion:** Leads to the false assumption that the build process failed or that the account permissions are incorrect.

## Correct Approach
Always use an official transport tool:
1.  **EAS CLI:** `eas submit --platform ios --path ./path.ipa` (Recommended for Expo).
2.  **Transporter App:** Official macOS app for drag-and-drop upload.
3.  **Xcode:** Using the `Archive > Distribute App` flow.
