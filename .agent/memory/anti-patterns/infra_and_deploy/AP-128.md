# AP-128 — Missing Specifier em Re-export de Workspace Sub-path

**Sintoma:** `Error: Missing "./schemas/X.js" specifier in "@dosiq/core" package` em Vite/Vitest.

**Causa:** `package.json` do workspace package tem `exports` field sem o sub-path wildcard. Vite 7 respeita o `exports` field mesmo quando um alias de path está configurado para o package.

**Fix:** Adicionar ao `package.json` do package:
```json
"./schemas/*": "./src/schemas/*.js"
```

**Regra associada:** R-213
