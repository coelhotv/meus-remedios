# [K-088] Path do GitHub CLI (gh) no macOS Homebrew

## Descrição
No ambiente macOS do usuário, a ferramenta `gh` é instalada via Homebrew e não é carregada automaticamente no shell não-interativo do agente.

## Localização
`/opt/homebrew/bin/gh`

## Solução para Agentes
Sempre exportar o path do Homebrew antes de executar comandos `gh`:
`export PATH="/opt/homebrew/bin:$PATH" && gh ...`
