# K-087 — Usuário de Homologação e Testes

## Contexto
Para evitar a modificação acidental de dados de produção (conforme R-179/AP-H28), o projeto utiliza uma conta dedicada para testes, homologação e submissão para lojas de aplicativos.

## Detalhes da Conta
- **Email**: `review@coelho.me`
- **Uso**: Testes de UI, automação de browser (Agent Browser), validação de fluxos de ponta a ponta.
- **Isolamento**: Esta conta contém dados fictícios e protocolos de demonstração que podem ser alterados/resetados sem impacto real à saúde dos usuários finais.

## Diretriz de Uso
Sempre que um agente precisar realizar um login via browser para validar se um medicamento aparece no dashboard ou se um botão está funcionando, DEVE utilizar primariamente esta conta.
