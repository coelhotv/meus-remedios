The custom polyfill in apps/mobile/polyfills.js (387 lines) is the DEFINITIVE URL solution for Hermes/Expo Go SDK 53. Both react-native-url-polyfill (NativeModules.BlobModule crash) and whatwg-url-without-unicode (WebIDL ctorRegistry crash) fail. The custom polyfill patches URL.prototype getters/setters and uses _searchPairs backing store.
