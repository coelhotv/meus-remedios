# [K-089] Layout Dashboard "Dona Maria" (Modo Simple)

## Descrição
O modo "Simple" é uma simplificação da aba Hoje para usuários senior ou com baixa carga de medicamentos, inspirado no perfil "Dona Maria".

## Características
1. **Lista Direta**: Não agrupa por turnos.
2. **Cronologia**: Ordenação estrita pelo horário da dose no dia.
3. **Fricção Zero**: O usuário não precisa abrir accordions para ver o que tomar em seguida.
4. **Visão Parcial**: Se houver mais de 3 medicamentos, o app forçará o modo `Complex` automaticamente.

Paridade com o dashboard da versão Web (Wave 10A).
