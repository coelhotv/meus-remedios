# CON-008: medicineSchema (Zod)

**File:** src/schemas/medicineSchema.js
**Type:** Database contract

## Schema Definition

```javascript
const medicineSchema = z.object({
  id: z.string().uuid(),
  name: z.string().min(1).max(255),
  laboratory: z.string().min(1),
  active_ingredient: z.string(),
  dosage_per_pill: z.number().positive(),
  dosage_unit: z.enum(['mg', 'mcg', 'g', 'ml', 'unidade']),
  type: z.enum(['comprimido', 'capsula', 'liquido', 'injecao', 'pomada', 'spray', 'outro']),
  price_paid: z.number().positive(),
  regulatory_category: z.enum(['Genérico', 'Similar', 'Novo']),
  user_id: z.string().uuid()
})
```
