# CON-001: medicineService.getAll()

**Status:** stable
**Type:** service-method
**File:** src/features/medications/services/medicineService.js

## Signature

```javascript
async getAll(): Promise<Medicine[]>
```

## Returns

```typescript
[
  {
    id: string (uuid)
    name: string
    laboratory: string
    active_ingredient: string
    dosage_per_pill: number (mg)
    dosage_unit: string ('mg', 'mcg', 'g', etc.)
    type: enum ('medicine' | 'supplement' | 'other')
    price_paid: number
    user_id: string (uuid)
    created_at: ISO 8601 timestamp
    stock: Stock[]  // Remaining inventory lots
    purchases: Purchase[]  // Immutable purchase history
    avg_price: number  // Weighted average from purchases
    regulatory_category: enum ('Genérico' | 'Similar' | 'Novo')
  }
]
```
