# R-022 — Dosage in Pills, Not Milligrams [CRITICAL]

## Rule
`quantity_taken` must be recorded in pills (Zod schema limit: 100), NEVER in milligrams.

## Why
Recording 2000mg exceeds Zod's `.max(100)` validation. The schema represents pill count, not dosage weight.

## Context
This is a domain-specific rule for meus-remedios. The `quantity_taken` field represents how many pills the patient took, not the total milligrams.
