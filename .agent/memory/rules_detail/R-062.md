# R-062 — Quality Over Speed [HIGH]

## Rule
One well-reviewed PR is worth more than 3 rushed PRs. Never sacrifice code review rigor for delivery speed. Gemini Code Assist review is non-negotiable.

## Why
2026-W09 coordination failure. Pressure to deliver faster led to skipping review steps. The resulting production bugs cost more time to fix than the review would have taken.
