# R-050 — JSDoc in Portuguese [MEDIUM]

## Rule
All JSDoc comments must be in Portuguese from the first commit.

## Template
```javascript
/**
 * Descrição da função.
 * @param {tipo} nome - Descrição do parâmetro.
 * @returns {tipo} Descrição do retorno.
 */
```

## Why
PR #44 review. The project is a Portuguese-language app for Brazilian users; code comments should be consistent with the domain language.

## Scope
All new JSDoc. Existing English JSDoc should be updated when touching the file.
