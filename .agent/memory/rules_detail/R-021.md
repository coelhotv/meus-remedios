# R-021 — Zod Enums in Portuguese [HIGH]

## Rule
All Zod schema enum values must be in Portuguese. English values cause UI inconsistencies. Always export label maps for UI display.

## Why
PR review catch — English enum values leaked into UI labels, causing inconsistent UX in a Portuguese-language app.

## Example
```javascript
// Correct
const FREQUENCIES = ['diário', 'dias_alternados', 'semanal', 'personalizado', 'quando_necessário']
export const FREQUENCY_LABELS = {
  diário: 'Diário',
  dias_alternados: 'Dias Alternados',
  semanal: 'Semanal',
  personalizado: 'Personalizado',
  quando_necessário: 'Quando Necessário'
}

// Wrong
const FREQUENCIES = ['daily', 'weekly']
```
