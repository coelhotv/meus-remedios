# R-026 — Consultation Summary Must Count Actual Doses Taken [HIGH]

## Rule
In consultation and PDF summaries:
- Use **dose-count metric** (`taken`) as the numerator for `taken/expected`
- Keep `takenAnytime` only for **secondary insights**

## Why
`takenAnytime` counts the presence of a log per protocol-day and inflates totals when a protocol has multiple time slots. This makes adherence look better than it is.

## Metric definitions
- `taken` — dose events where the patient confirmed taking at the scheduled time
- `takenAnytime` — any log entry for that protocol day, regardless of time
