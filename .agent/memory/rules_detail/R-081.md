# R-081 — Test Suite Performance Strategy [HIGH]

## Rule
Choose the correct test command for the context:

| Context | Command | Threads | Duration |
|---------|---------|---------|----------|
| Development (default) | `npm run test:fast` | 1 | ~6.5 min |
| Development (parallel) | `npx vitest run --maxThreads=2` | 2 | ~3-4 min |
| Low-RAM (8GB) | `npm run test:lowram` | sequential | ~20 min |
| **Agent sessions** | `npm run validate:agent` | 4+ | <10 min |
| CI/CD | `npm run validate:full` | 4+ | <5 min |

## Agent mandatory
Agents MUST use `validate:agent` — see R-074.

## Decision matrix
- Local dev → `test:fast` (recommended)
- Need speed → `--maxThreads=2` (caution: race conditions possible)
- 8GB machine → `test:lowram` (before push only)
- Agent/CI → `validate:agent` / `validate:full`
