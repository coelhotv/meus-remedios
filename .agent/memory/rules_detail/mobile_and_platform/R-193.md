# R-193 — React Navigation createNavigationContainerRef

## Rule

Para suportar deep links em cold start (app não estava aberto), usar `createNavigationContainerRef()` em vez de listener fallback manual:

```js
// ✅ CORRETO
const navigationRef = createNavigationContainerRef()

function navigate(name, params) {
  navigationRef.isReady() ? navigationRef.navigate(name, params) : null
}

<NavigationContainer ref={navigationRef}>...</NavigationContainer>
```

Elimina necessidade de `getLastNotificationResponseAsync()` + `useEffect` fallback complex.

## Why

- **API Official**: React Navigation oficial documenta `createNavigationContainerRef` exatamente para este caso
- **AP-116 Prevention**: Evita reexecução redundante em logout+login
- **Mobile Performance**: Menos listeners apensos, menos memory bloat em lifecycle

## How To Apply

Ao implementar usePushNotifications hook:
1. Mover `navigationRef` para módulo-level (não criar nova instância por render)
2. Usar `navigate()` helper que checa `.isReady()` antes de dispatch
3. Remover listener fallback completo
