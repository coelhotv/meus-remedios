# R-074 — Use validate:agent for Agent Sessions [HIGH]

## Rule
When running tests from an agent session, ALWAYS use `npm run validate:agent`.

Never use `validate:quick` or `validate:full` from an agent session.

The wrapper enforces a 10-minute kill switch. If the suite needs >10 minutes, the tests are broken — fix them, don't raise the timeout.

## Exit codes
- `0` — Tests passed
- `1` — Tests failed
- `124` — Timeout exceeded (killed after 10min)

## Why
Agents running unbound test suites can consume CI minutes and hang indefinitely. The kill switch is a safety net, not a workaround.
