# R-078 — Aggressive Test Cleanup [HIGH]

## Rule
Every test's `afterEach` must call all four cleanup operations:

```javascript
afterEach(() => {
  clearCache()          // or clearState() — project-specific
  vi.clearAllMocks()
  vi.resetAllMocks()
  vi.clearAllTimers()
  if (global.gc) global.gc()  // optional: hint GC on low-RAM machines
})
```

## Why
2026-02-23 test isolation. Shared state between tests caused intermittent failures that were hard to reproduce. Aggressive cleanup ensures each test starts from a clean slate.
