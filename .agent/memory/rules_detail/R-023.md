# R-023 — Operation Order for Doses [HIGH]

## Rule
Always follow this order for dose registration:
1. **Validate** — check stock, schema, constraints
2. **Record** — create the dose log
3. **Decrement** — decrease stock

Never decrement stock before validating and recording.

## Why
If recording fails after decrementing, stock is incorrect without a corresponding log. The validate-first order ensures atomicity of intent.
