# R-011 — LogForm Dual Return Type [HIGH]

## Rule
`LogForm` returns:
- **Array** when `type === 'plan'` (bulk logging)
- **Object** when `type === 'protocol'` (single protocol log)

Always check `Array.isArray(logData)` before processing the result.

## Why
Processing both return types the same way causes silent data loss or runtime errors.

## Pattern
```javascript
const logData = await LogForm(type, data)
if (Array.isArray(logData)) {
  // bulk plan processing
} else {
  // single protocol processing
}
```
