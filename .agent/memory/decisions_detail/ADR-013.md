# ADR-013: Shadow System: Ambient Shadows Over Neon Glows

**Status:** Accepted | **Date:** 2026-03-24 | **Source:** PR #417
**Tags:** design, shadows, tokens

## Context
Neon glow effects are visually jarring and inappropriate for healthcare application.

## Decision
Replace neon glows with Material Design 3 ambient shadow system for depth and hierarchy.

## Implementation
```css
--shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.05);
--shadow-md: 0 4px 6px rgba(0, 0, 0, 0.1);
--shadow-lg: 0 10px 15px rgba(0, 0, 0, 0.15);
--shadow-xl: 0 20px 25px rgba(0, 0, 0, 0.2);
```

## Consequences
✅ Elegant depth perception
✅ Professional appearance
✅ Better contrast for accessibility
⚠️ Requires careful implementation (no glows)
⚠️ Different visual aesthetic
