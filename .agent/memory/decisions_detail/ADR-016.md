# ADR-016: Calendar-Driven History Navigation

**Status:** Accepted | **Date:** 2026-03-28 | **Source:** PR #437
**Tags:** ux, history, calendar

## Context
Users struggled with chronological history navigation, especially for retrieving entries from specific dates.

## Decision
Implement calendar-driven history view with date picker and month/week navigation for quick access to past entries.

## Consequences
✅ Much faster historical lookups
✅ Better visual organization
✅ Reduced cognitive load
⚠️ Requires calendar component
⚠️ Mobile optimization needed
