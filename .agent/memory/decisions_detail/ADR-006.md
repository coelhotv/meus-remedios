# ADR-006: Claude Code GitHub Actions Integration

**Status:** Accepted | **Date:** 2026-03-21 | **Source:** PR #414
**Tags:** ci-cd, automation, claude-code

## Context
Need automated code review and validation in GitHub Actions to catch issues early and maintain code quality.

## Decision
Integrate Claude Code GitHub Actions workflow for automated validation on every PR.

## Implementation
- Add Claude Code workflow to `.github/workflows/`
- Run on: PR open, commits pushed, ready for review
- Check: linting, tests, type safety, security
- Post results as PR comment

## Consequences
✅ Automated quality gates
✅ Consistent feedback across team
✅ Catches issues before human review
⚠️ Requires Claude Code API setup
⚠️ May need tuning to reduce false positives
