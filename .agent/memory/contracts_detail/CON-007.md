# CON-007: parseLocalDate(dateString)

**File:** src/utils/dateUtils.js

## Signature

```javascript
function parseLocalDate(dateString: string): Date
```

## Purpose

Parse YYYY-MM-DD to Date object correctly for BR timezone (GMT-3).

## CRITICAL RULE

**NEVER** use `new Date('YYYY-MM-DD')` — it parses as UTC midnight.

## Consumers

- `src/features/stock/services/stockService.js` — expiry dates
- `src/features/adherence/services/adherenceLogic.js` — dose dates
- `src/features/reports/services/pdfGeneratorService.js` — report dates
- `src/views/HealthHistory.jsx` — timeline dates
