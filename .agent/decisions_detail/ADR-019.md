# ADR-019: PDF Report: Clinical Consultation Format

**Status:** Accepted | **Date:** 2026-03-24 | **Source:** PR #421
**Tags:** reports, pdf, feature

## Context
Healthcare professionals need to generate standardized reports from consultations for medical records.

## Decision
Implement PDF generation with clinical consultation format:
- Patient demographics
- Medications and dosages
- Recent adherence data
- Clinical notes
- Print-friendly layout

## Implementation
- Use `pdf-lib` or `pdfkit` for generation
- Template system for consistent formatting
- Download/email delivery

## Consequences
✅ Professional documentation
✅ Better medical record keeping
⚠️ Complex PDF generation logic
⚠️ Testing requires visual validation
