# ADR-015: Medicines View: Consultation Mode with Read-Only Interface

**Status:** Accepted | **Date:** 2026-03-31 | **Source:** PR #440
**Tags:** medicines, ux, consultation

## Context
Healthcare professionals need to review patient medicines during consultations without risk of accidental modifications.

## Decision
Implement consultation mode in medicines view with:
- Read-only interface (no edit/delete buttons)
- Modal presentation for focus
- Print option for prescriptions
- Export to PDF

## Implementation
- `useConsultationMode()` hook
- Conditional rendering of action buttons
- Read-only form fields

## Consequences
✅ Safety for clinical use
✅ Better focus during consultations
⚠️ Must verify button visibility logic
⚠️ UX complexity increases
