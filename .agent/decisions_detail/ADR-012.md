# ADR-012: No Sharp Borders: Minimum 0.75rem Border Radius

**Status:** Accepted | **Date:** 2026-03-24 | **Source:** PR #417
**Tags:** design, ui-guidelines

## Context
New design aesthetic requires softer, more approachable appearance appropriate for healthcare app.

## Decision
All UI elements have minimum 0.75rem border radius:
- Input fields, buttons: 0.75rem
- Cards, modals: 1.5rem
- Large containers: 2rem (--radius-card)
- No sharp corners anywhere

## Implementation
```css
--radius-sm: 0.75rem;
--radius-md: 1.5rem;
--radius-card: 2rem;
--radius-lg: 3rem;
```

## Consequences
✅ Softer, more friendly appearance
✅ Consistent design language
⚠️ Slightly larger visual footprint
⚠️ May not suit all design scenarios
